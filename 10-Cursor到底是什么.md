# 第10课：Cursor 到底是什么？

## 先说结论

一句话：

> Cursor = 集成了 AI Agent 的编程工作台（IDE）。

---

# 什么是 IDE？

IDE 全称：

Integrated Development Environment

中文：

集成开发环境

---

可以理解成：

> 程序员的工作台。

---

# Cursor 的本质

Cursor 不是模型。

也不是单纯的 Agent。

更准确地说：

Cursor = AI 编程工作台。

---

# Cursor 和 Claude Code

Claude Code：

- 读代码
- 改代码
- 测试代码

更像程序员。

---

Cursor：

- 展示代码
- 管理文件
- 调用AI
- 组织工作流程

更像办公室。

---

所以：

Claude Code 可以在 Cursor 中工作。

---

# Cursor 为什么流行

原因：

1. 简单

2. 学习成本低

3. AI体验优秀

---

# Cursor 和 Codex

Codex：

- 更像程序员

Cursor：

- 更像程序员办公室
